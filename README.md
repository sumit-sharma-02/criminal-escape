## Quick Start

```shell
$ git clone https://github.com/sumit-sharma-02/criminal-escape.git
$ cd criminal-escape
$ npm install
$ cd backend
$ npm run dev
```

Open another terminal
```shell
$ cd frontend
$ npm install
$ npm start
```
